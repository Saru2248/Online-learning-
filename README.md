# 🎓 EduFlow — Online Learning & Course Recommendation Platform

A full-stack EdTech platform inspired by Coursera & Udemy that provides **personalized course recommendations, progress tracking, and a modern learning experience**.

---

## 🚀 Key Highlights

* 🔐 JWT-based authentication system
* 📚 Course creation, enrollment, and progress tracking
* 🔍 Full-text course search with filters
* 🤖 AI-based course recommendation engine
* 📊 User dashboard with learning analytics
* 🐳 Dockerized setup for easy deployment

---

## 🧠 What I Built

* Designed and developed a **full-stack application** using:

  * Next.js (frontend)
  * NestJS (backend API)
  * FastAPI (ML recommendation service)
* Implemented **hybrid recommendation system**:

  * TF-IDF (content-based)
  * Collaborative filtering
* Built scalable architecture using:

  * PostgreSQL (database)
  * Redis (caching)
  * Meilisearch (search engine)

---

## 🏗️ System Architecture

```text
Frontend (Next.js)
        ↓
Backend API (NestJS)
        ↓
Database (PostgreSQL)
        ↓
ML Service (FastAPI)
```

---

## ⚙️ Tech Stack

* **Frontend:** Next.js, Tailwind CSS
* **Backend:** NestJS, Node.js
* **Database:** PostgreSQL
* **ML Service:** Python, FastAPI, scikit-learn
* **Tools:** Docker, Redis, Meilisearch
